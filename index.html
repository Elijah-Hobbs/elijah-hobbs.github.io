C++ NACA Airfoil Generator

This project was created in C++ and creates a window that allows the user to input a 4 digit NACA airfoil code and view the airfoil it corresponds with. The project uses GLFW to create the window, OpenGL to display the airfoil geometry within the window, and ImGUI to create UI elements and process inputs. The project uses the standard NACA equations to calculate an airfoil's geometry, camber line, and chord line according to the user's inputs.

Turbine Based Combined Cycle Jet Engine Simulation

This project was created in MATLAB and simulates a series of turbine based combined cycle jet engine with varying properties. The program varies a series of critical parameters like the pressure ratio, the speed at which the engine switches from its turbojet stage to its ramjet stage, and the maximum temperature in both the turbojet and ramjet nozzles to see what values of the properties result in the most efficient engine. After a simulated engine's properties are determined a cycle analysis is performed a series of efficiency characteristics like the overall efficiency and the thrust specific fuel consumption are calculated. The most effient engine's properties are displayed after all engines have been analyzed.

Python NACA Airfoil Generator

This project was created in Python and the program takes in an input of a 4 digit NACA airfoil number and uses it to graph the associated airfoil. A large number of points are generated based on the standard equations for the airfoil's geometry, camber line, and chord line. After all of the relevant points have been generated they are plotted in a new window using PyOpenGL.

Diving Physics Simulation

This project was created in MATLAB and simulates how a diver moves when they are diving from a platform. The simulation uses a differential equation solver in MATLAB and steps through time simulating the divers motions as they change their moment of intertia in order to flip in the air. After finishing the simulation the program will create a few graphs and a short video of the dive using MATLAB's plotting functions.

Intermediate Axis Theorem Simulation

This project was created in MATLAB and shows an example of the intermediate axis theorem simulated in MATLAB. It shows how the same object will rotate stably about its most and least significant axes and rotate unstably about its intermediate axis.
