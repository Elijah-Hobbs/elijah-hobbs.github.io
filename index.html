NACA Airfoil Generator

This project was created in Python and the program takes in an input of a 4 digit NACA airfoil number and uses it to graph the associated airfoil. A large number of points are generated based on the standard equations for the airfoil's geometry, camber line, and chord line. After all of the relevant points have been generated they are plotted in a new window using PyOpenGL.

Diving Physics Simulation

This project was created in MATLAB and simulates how a diver moves when they are diving from a platform. The simulation uses a differential equation solver in MATLAB and steps through time simulating the divers motions as they change their moment of intertia in order to flip in the air. After finishing the simulation the program will create a few graphs and a short video of the dive using MATLAB's plotting functions.

Intermediate Axis Theorem Simulation

This project was created in MATLAB and shows an example of the intermediate axis theorem simulated in MATLAB. It shows how the same object will rotate stably about its most and least significant axes and rotate unstably about its intermediate axis.
